## Sobre o Projeto
Desenvolver uma página destinada a novas contratações de uma empresa de TI.

O projeto tem como proposta, concluir durante o período de 7 dias, sendo cada dia uma seção da página para produzir. Podendo ser usado para testar as habilidades em HTML e CSS e aprender novas funcionalidades.

> Este projeto foi concluído durante o desafio de **#7DaysOfCode - HTML e CSS** promovido pela **Alura**.

## Detalhes de Implementação

Embora o projeto tenha seguido fielmente a estrutura proposta no Figma da Alura, tomei liberdade para adicionar algumas interatividades da página com o cursor.

- **Interatividades adicionadas:** Inseri animações de 'hover' nos botões e cards de vagas, utilizando 'transition' e 'transform'.

## Tecnologias utilizadas 

- **HTML:** Estruturação semântica da página.
- **CSS:** Estilização e layout.
- **Google Fonts:** Importação de fontes para fidelidade ao figma.
